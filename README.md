# Yoga Webinar Template

A responsive webinar landing page for yoga classes built with HTML, Bootstrap, CSS, and JavaScript.

## Features
- Responsive hero banner section
- Video gallery section
- Yoga class listing section
- Testimonials section
- Registration form with validation
- Firebase-ready registration storage with local fallback

## Structure
- index.html
- assets/css/style.css
- assets/js/api.js
- assets/js/script.js
- assets/js/firebase.js

## Run locally
Open index.html in a browser, or serve the folder with a simple static server.
